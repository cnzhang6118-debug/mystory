# 《仙路烟尘》世界观设定

> 数值与名词的唯一权威来源。正文用到的术语、价格、地名一律以本文为准；新增设定先写进这里再进正文。

## 一、境界体系（经典骨架 + 自创细化）

| 大境界 | 内部划分 | 寿元上限 | 备注 |
|---|---|---|---|
| 凡人 | — | 约七十 | |
| 炼气 | 一至九层 + 圆满 | 约一百二十 | 一至三层「感气」、四至六层「引气」、七至九层「盈窍」，圆满者方有筑基之望 |
| 筑基 | 初期／中期／后期／圆满 | 约二百五十 | 须过「筑基三关」：引灵、固府、铸台。铸台铸的是什么，各家说法不一（※手札批注独见，深伏笔） |
| 金丹 | 九品，九品最低 | 约六百 | 丹品由筑基之"台"与结丹之"引"共同决定，结丹后终身不改 |
| 元婴 | 初／中／后 | 约一千二 | 州府大宗底蕴，边郡传说 |
| 化神 | — | 不详 | 本方修仙界凤毛麟角，化神之后留白 |

**力量铁律**：炼气期内，层差可借预谋、外物、地形弥补；筑基对炼气是结构性碾压（灵压可直接锁住炼气低层行动）；金丹对筑基同理。

## 二、灵根与资质

- 五行灵根，以纯杂论高下：天灵根（单属性）＞双灵根＞三灵根＞四五行驳杂（下品杂灵根）。
- 杂灵根引气慢、瓶颈多，坊市散修十之七八皆是，宗门不收，只能自谋出路——这是散修生态的根源。
- **杜仲：五行驳杂、水木稍显，下品杂灵根。**

## 三、货币与物价（第一卷基准）

| 项目 | 价 |
|---|---|
| 一块下品灵石 | ≈ 凡人白银八十至一百两 |
| 灵石进阶 | 一百下品 = 一中品（边郡罕见中品） |
| 聚气散一瓶（十粒，炼气一至五层辅助） | 四块下品 |
| 坊市摊位一日租金 | 半块下品（可以银钱折抵） |
| 散修一年基本开销（含修炼） | 三十至五十块下品 |
| 凡人五口之家一年嚼用 | 约二十两银 |

**经济常识**：炼气散修常年处于"修炼一停就饿死，一修炼就破产"的循环。灵石用于打坐是奢侈，多数人攒着买丹、买符、买命。

## 四、地理（第一卷视野）

- **大黎王朝**：凡人王朝，当朝年号「承安」。官府不知仙门底细，仙凡两界默契互不干涉。
- **陇阳郡**：大黎边郡，郡城陇阳城。山多田薄，药材集散之地。
- **青溪镇**：陇阳郡边缘山镇，药户聚居，背靠苍岭。镇上济生堂是百年药铺。
- **苍岭**：绵延大山，深处有妖兽，外缘是药户讨生活的地界。由外向内：**老鸦坡**（外山，熟路）→ **黑水涧**（中山溪涧，水青黑）→ **涧源阴谷**（七叶参群落所在，已是鹰愁崖地界边缘）。**鹰愁崖**：苍岭险地，杜父坠亡处，洞府亦在此崖裂罅中（承安十二年山洪震裂出罅口，后再次塌方掩埋）。山脚有药户乱坟岗，杜父葬于此。
- **青蚨坊**：陇阳城西四十里黑石坳中的散修坊市，白石李氏经营，三教九流汇集。入口与暗号每隔数年更换（※手札所记为三百年前旧制，已失效）。
- 郡外：州府「云川州」，有元婴大宗坐镇（第一卷只闻其名）。

## 五、势力（陇阳郡格局）

| 势力 | 层次 | 概况 |
|---|---|---|
| 敛芳谷 | 金丹小宗（药修） | 郡内药材生意的上游，金丹老祖坐镇传说中。外门执事多贪，内外门泾渭分明 |
| 白石李氏 | 筑基家族 | 经营青蚨坊，代管敛芳谷的七里坡药园——代管关系是两家结构性矛盾所在 |
| 散修群体 | 炼气为主 | 坊市底层生态：供役、采药、跑山、互坑 |

## 六、杜仲的遗产清单（洞府所得，全书账本）

1. **《青元诀》批注本**——功法本身是烂大街的炼气功法，价值全在老人一生经验的批注。其中「铸台」一节批注语焉不详、似有撕改痕迹（深伏笔，与筑基三关呼应）。
2. **下品灵石二十七块**——一个散修一生谨慎，最后剩下的全部家当。
3. **半旧储物袋一只**——容积一丈见方，炼气可用。
4. **短剑「青竹」**——下品法器，剑身有旧伤。
5. **青铜小钟**——无名，防御法器，钟光护身远超下品法器水准。**动用时钟身古纹显现，识货者可辨其来历（全书后患，手札明语：「钟非常物，非死生之际勿出」）。**
6. **符笔一支、黄符纸若干、旧符三张**（火蛇符二、土墙符一）。
7. **手札一本**——无署名，卷边，最后几页被主人亲手撕去。

## 七、杜仲的凡人手艺（非超自然，长期资产）

- **辨药眼力**：济生堂帮工 + 药户家传，对草木药性的直觉远超同侪。
- **识字记账**：能读手札、看穿契约陷阱、左手写匿名字条。
- **药户手艺**：自制「断肠烟」（迷目药粉）、蛇虫防治、山中辨路设套——他的真正底牌。

## 八、术语小辞典（随写作增补）

- **供役／供户**：仙门向散修、凡人药户发包的原料供应差事，给灵石或残丹。
- **客作**：坊市产业雇佣的长期散修工。
- **黑话举例**：坊市称灵石为「青蚨」，称官面身份为「壳」，称被仙门记名为「上册」。
- **七叶参**：凡品药材（非灵药），喜阴湿，根茎带"老疙瘩"者为陈货。平价百文上下一株，承安十二年春因城中急症涨至二百文。
- **仁和号**：陇阳城凡人药行，与济生堂有往来。
- **杜家三规矩**（杜父用命换的，正文反复回访）：一人不进鹰愁崖；水浑不过涧；雷闷不登高。
