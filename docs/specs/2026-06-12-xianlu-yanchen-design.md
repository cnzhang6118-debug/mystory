# 《仙路烟尘》整体设计稿

> 状态：已与作者确认全部核心决策（2026-06-12）
> 本文档是全书的最高设计依据。与本文冲突的后续设定，须先修订本文再落地。

## 一、作品定位

凡人流修仙长篇，目标 100 万字以上，平台连载规格。

一个边陲药户少年得了一位无名散修的遗泽，在冷酷现实的修仙界里步步求活、缓缓向上。金手指只负责"让他入局"和"教他别死"，路全靠他自己走——心性、成长、运气、把握机会的能力，才是他真正的依仗。

| 决策项 | 结论 |
|---|---|
| 篇幅 | 100 万字以上，按卷推进 |
| 文风 | 凡人流（生存现实主义），情感细腻，文戏对标《凡人修仙传》动画版 |
| 章节规格 | 每章 4000~6000 字，章末留钩，前三章抓人；节奏服从故事本身，铺垫可以从容 |
| 境界体系 | 经典骨架（炼气→筑基→金丹→元婴→化神）+ 大境界内部自创细化 |
| 金手指 | 一次性机缘 + 遗泽（前人手札），无持续性超自然外挂 |
| 主角性格 | 谨慎但有温度，越往上走对凡人的感观越深刻 |
| 协作模式 | 卷级详细章纲先行 → 逐章写作（每次 1~2 章）→ 反馈修订 → 记账更新 |

**书名定为《仙路烟尘》**：烟尘既是他一步步走远的仙路上的风尘，也是他回头望时故乡的人间烟火。曾考虑《无名碑》，因悬念有保质期（名字揭晓后书名即失效）而弃用；但"无字碑"情节本身保留，作为中期重磅情感节点。

## 二、主角：杜仲

- **出身**：大黎王朝边郡青溪镇，药户人家。镇上规矩，药户的孩子取药名好养活——杜家二小子，就叫了杜仲。哥哥早夭，父亲采药坠崖亡故，他十四岁起独自进山采药，养着体弱的母亲和年幼的小妹。
- **识字的来历**（关键前置设定，不可遗漏）：自小在镇上济生堂药铺帮工，掌柜教他认药、认字、记账——所以他读得懂手札，也练出辨药的底子。这是他日后在坊市立足的两样本钱。
- **性格**：谨慎是被生活磨出来的（山里走错一步就是他爹的下场），温度是责任给的（身后有娘和小妹）。不天真，但没坏；会算计，但有底线。
- **资质**：下品杂灵根，五行驳杂、水木稍显。修行路上他永远是"资质平平的那一个"。
- **成长母题**：引路人母题——手札教他活下去 → 他走到手札主人止步之处，从此无人引路 → 他最终成为后来者的"手札"。

## 三、手札主人（全书"第二主角"）

一具坐化的骸骨，一本没有署名的手札。**主角从头到尾不知道他的名字**，只能从字里行间拼凑这个人的一生：凡人佃户出身，自己摸到修行门槛，散修一生谨慎到近乎苛刻，止步筑基巅峰，晚年回到故乡青溪镇的山中隐居、坐化。

- **无字碑之诺**：杜仲安葬骸骨，立无字碑——"等我知道您的名字，再来补上。"中期（约金丹前后）查明真名并刻碑，是该阶段的情感峰顶；刻碑不是终点，名字背后牵出的旧史才是更深的线。
- **遗产四件**：
  1. 一部完整的入门功法——大路货，但布满老人一生经验的批注。**功法是大路货，批注是孤本**。其中关于"筑基筑的是什么"的独见，是主角日后根基异于常人的深伏笔。
  2. 半袋灵石——少到必须精打细算。
  3. 几件旧法器——**其中一件的来历是后患**，动用会被识货者认出，卷二之后的追兵线由此引爆。
  4. 手札本身——散修从凡人到筑基巅峰的一生记录：怎么省灵石、哪种活计碰不得、坊市黑话、哪些人笑着递刀。不给力量，只给经验。
- **手札的边界（防无敌设计，必须严守）**：
  - 主人止步筑基——金丹之后的事，手札一个字也给不了；
  - 信息过时几百年——坊市格局、宗门、物价全变了，指引要校正着用，**有时还会反过来坑主角**（第一卷即安排：手札说信誉可靠的老字号，如今早已易主）；
  - 金手指本身也是冲突来源，不只是福利。
- **死因与撕页（全书第一暗线 L1）**：手札最后几页是他自己撕掉的。年轻时卷入一件事——某段被抹去的修仙界旧史，看了不该看的东西，被追杀，隐姓埋名几百年，暗伤至死。撕掉的页、藏起的东西、还在找他的势力，贯穿全书，末卷收口。

## 四、世界观与境界

- **地理纵深**（随境界展开，第一卷只见最底层）：
  青溪镇（凡人，情感锚点，每卷回望一次）→ 郡城与散修坊市（第一卷主舞台）→ 周边小仙门（筑基、金丹层次）→ 州府大宗（元婴，遥远的传说）→ 更远处留白。
- **境界体系**：炼气（九层+圆满）→ 筑基（初/中/后/圆满）→ 金丹（分品）→ 元婴 → 化神，化神之后留白。自创细化集中在大境界内部门槛（"筑基三关""金丹品级成因"等），细表落 `bible/worldbuilding.md`。
- **修为曲线基准**：第一卷结束时炼气四层。够慢，够凡人流。后续每卷的修为增幅在卷纲阶段单独核定，严禁超速。

## 五、暗线与伏笔架构

| 层级 | 内容 | 跨度 |
|---|---|---|
| L1 全书最深线 | 手札主人当年卷入的旧史真相：撕掉的页、藏起的东西、追索的势力 | 第一卷埋下，末卷收口 |
| L2 跨卷线 | ① 无字碑之诺（查名→刻碑）② 旧法器暴露引来的追兵 ③ 功法批注中"道基"之秘 → 筑基异象 | 各跨 2~4 卷 |
| L3 卷级线 | 每卷自身的阴谋/争夺主线 | 卷内闭合 |
| L4 情感暗线 | 青溪镇对照组：母亲、小妹、济生堂掌柜——每次回乡的物是人非，"脱离凡人"的刻度尺 | 全书 |

所有伏笔自第一天起登记 `bible/plot-threads.md` 台账，字段：编号/内容/埋点章节/计划回收卷/状态（未回收/推进中/已回收）。**台账里"未回收"条目超过 15 条时必须先收再埋。**

## 六、第一卷《山中得卷》弧线（约 40 章，16~24 万字）

1. **黄金三章（1~3）**：山洪 → 洞府得物 → 安葬立碑、初读手札、第一缕灵气。凡人生活铺垫压缩进第 1 章前半，其余靠后续闪回。
2. **前段（4~12）**：白天采药养家、夜里偷偷修行的两头烧；按手札指引初探郡城坊市；吃手札过时的第一个亏（老字号易主）。
3. **中段（13~28）**：凭辨药眼力和手札黑话在坊市谋活计；卷入一桩围绕药园的小阴谋；**第一次杀人**——全卷文戏重心，对照手札里老人写自己第一次杀人的那一页。
4. **高潮（29~37）**：阴谋收口、生死险境，被迫动用旧法器——被某人记住了。
5. **尾声（38~40）**：回乡。小妹长高了，镇上有老人走了，他对娘撒了第一个谎。卷末钩：有人在打听山洪之后那片山。

## 七、写作风格规范（要点，细则落 style-guide.md）

- 凡人流叙事纪律：资源有价、实力有数、危险真能死人；"恰好"全书限量使用。
- 文戏标杆：《凡人修仙传》动画版改编水准——克制、留白、用细节而非抒情堆砌写情感。
- 主角视角为主，严禁全知视角剧透暗线。
- 每章末必须有钩子；章内至少推进一条线（情节/修为/人物/伏笔，四选一以上）。

## 八、记忆系统与项目结构

```
test/                          # 项目根（git 仓库）
├── CLAUDE.md                  # 写作守则：会话装载协议 + 记账协议（自动加载）
├── docs/specs/                # 本设计稿
├── bible/
│   ├── style-guide.md         # 文风细则
│   ├── worldbuilding.md       # 境界/功法/货币/地理/势力
│   ├── outline-master.md      # 全书总纲（分卷一句话）
│   ├── outline-vol1.md        # 第一卷卷纲 + 逐章梗概
│   ├── plot-threads.md        # 伏笔台账
│   ├── timeline.md            # 大事年表
│   └── characters/
│       ├── _index.md          # 人物总览：active / inactive / dead
│       └── <每人一档>.md
├── chapters/v1/               # 正文，001-章名.md
└── summaries/v1.md            # 每章 100~200 字摘要
```

- **人物管理**：只改状态、不删档案。active 进上下文全档，inactive 在 _index 留一行摘要，dead 留档不加载。
- **文件是唯一事实来源**：任何会话（包括失忆的新会话）仅凭文件即可完整恢复创作状态。

## 九、协作工作流

1. 作者确认本设计稿 →
2. 搭脚手架：写全套 bible 文件（风格指南/世界观/总纲/初始人物档案/伏笔台账初始登记）→
3. 第一卷逐章梗概（含每章钩子与伏笔埋点标记），交作者确认 →
4. 逐章写作循环：写 1~2 章 → 作者反馈 → 修订 → **记账**（更新摘要/人物/伏笔/时间线）→ 下一章。
5. 每卷末：卷级复盘 + 下一卷卷纲与章纲。
